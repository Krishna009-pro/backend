# Node.js Learning Repository

This repository contains projects, and lecture materials for learning backend development, covering topics such as Node.js, Express.js, MongoDB, MVC architecture, SQL, and REST APIs.

## Course Structure

- **Lecture - 1 & 2**: Basics of Node.js
- **Lecture - 3**: Express Js Fundamentals
- **Lecture - 4**: styling and UI with Tailwind Css
- **Lecture - 5**: Model-View-Controller (MVC) Architecture
- **Lecture - 6**: SQL Databases
- **Lecture - 7**: Cookies and Session Management
- **Lecture - 8**: File Upload and Download
- **Lecture - 9**: REST API and JSON Requests
- **Demostrating MERN stack**: Example MERN projects
- **Code By AI**: AI assisted code and examples

## Getting Started

To run the code in any specific lecture folder:

1. Navigate to the lecture directory:
   ```bash
   cd "Lecture - 5 - MVC Architecture"
   ```
2. Install dependencies (if a `package.json` is present):
   ```bash
   npm install
   ```
3. Run the application (typically with node or nodemon):
   ```bash
   node index.js
   # or
   npm run dev
   ```
